## 📌 What is this project?
An interactive Business Intelligence Dashboard built on the
Sample Superstore dataset (9,994 records · 2016–2019).
Covers full pipeline from Python data analysis to live web dashboard.

---

## ✨ Features
- 📊 4 Interactive charts
- 🎯 Animated KPI counters
- 🌙 Dark / Light mode
- 🔍 Live search & sort table
- ⬇ Export charts as PNG
- 📱 Mobile responsive
- 💫 Splash screen & toast alerts
